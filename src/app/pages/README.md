# Pages

Директория содержит все страницы приложения.

## Структура

- `auth-page-wrapper.tsx` - обертка для страниц авторизации (вход/смена пароля)
- `history-page.tsx` - страница истории обслуживания автомобиля
- `main-page.tsx` - главная страница приложения с навигацией
- `index.ts` - центральная точка экспорта всех страниц

## Использование

```tsx
import { AuthPageWrapper, MainPage, HistoryPage } from './pages';
```
